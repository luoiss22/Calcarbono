2. Uso del Token para Acceder a la API

Para usar el token en tus solicitudes, debes incluirlo en el encabezado de autorización:

Authorization: Bearer eyJ0eXAiOi... (tu token JWT completo)

Opciones para Usar tu Token:

  1. Navegador con Extensión: Instala una extensión que te permita modificar encabezados HTTP (como "ModHeader" para Chrome).
  2. Herramientas como Postman: Incluye el token en la sección de autenticación.
  3. Usar el Probador Integrado: Utiliza el navegador de API interactivo de Django REST Framework en la interfaz web.
  4. JavaScript: Prueba los endpoints directamente desde esta página.

3. Endpoints Disponibles
